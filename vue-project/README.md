# vue-project
vue3的后端管理系统
技术栈使用
vue3+组合式api
vite 构建工具
typeScript
vue-router
pinia
element-ui
axious

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
